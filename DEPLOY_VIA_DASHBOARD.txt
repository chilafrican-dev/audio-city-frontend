# Deploy Worker via Cloudflare Dashboard (Easiest Method)

## Steps:

### 1. Go to Cloudflare Dashboard
1. Open: https://dash.cloudflare.com
2. Log in to your account

### 2. Find Your Worker
1. Click: **"Workers & Pages"** in the left sidebar
2. Find: **"audio-city-api-d1"** (your worker name)
3. Click on it

### 3. Edit the Code
1. Click: **"Edit code"** or **"Quick edit"** button
2. You'll see a code editor

### 4. Copy Worker Code
1. Open `worker-d1.js` file from your computer
2. Select ALL the code (Cmd+A / Ctrl+A)
3. Copy it (Cmd+C / Ctrl+C)

### 5. Paste and Deploy
1. In the Cloudflare editor, select all existing code
2. Paste the new code (Cmd+V / Ctrl+V)
3. Click: **"Save and Deploy"** button (top right)
4. Wait for deployment to complete (usually 10-30 seconds)

### 6. Verify
After deployment, the new mastering endpoints will be active!

## What Gets Deployed:

- ✅ New mastering endpoints pointing to persistent tunnel
- ✅ Removed old VPS proxy code
- ✅ Updated to use `env.MASTERING_SERVER_URL` variable

## After Deployment:

The worker will use:
- `MASTERING_SERVER_URL` = `https://mastering.audiocity-ug.com` (already set)
- New endpoints: `/api/quick-master` and `/api/master-status/:id`
