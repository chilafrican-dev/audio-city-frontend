STEP 1: Deploy Frontend (5 minutes)
────────────────────────────────────
1. Go to https://www.netlify.com
2. Sign up (free, use GitHub/Google)
3. Drag & drop your "online master" folder
4. Wait 30 seconds
5. Done! You get: https://your-site-123.netlify.app

STEP 2: Deploy Backend (30 minutes)
─────────────────────────────────────
1. Get free VPS from Alavps.com
2. Follow instructions in: HOW_TO_DEPLOY.txt
3. Upload backend code
4. Install Node.js, FFmpeg, PM2, Nginx
5. Start server
6. Update frontend API URL
