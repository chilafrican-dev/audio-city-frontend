## üöÄ Deploy Options:

### Option 1: Via Cloudflare Dashboard (Easiest)
1. Go to: https://dash.cloudflare.com
2. Workers & Pages ‚Üí Create ‚Üí Create Worker
3. Name: `audio-city-api-proxy`
4. Copy entire contents of `worker.js` into editor
5. Click "Save and Deploy"

### Option 2: Via CLI (After fixing SSL)
```bash
npx wrangler login
npx wrangler deploy --config wrangler.workers.toml
```

## ‚öôÔ∏è After Deployment:

### 1. Set Backend URL
In Cloudflare Dashboard:
- Workers & Pages ‚Üí audio-city-api-proxy
- Settings ‚Üí Variables
- Add: `BACKEND_URL` = `https://api.audiocity-ug.com`
  (or your backend HTTPS URL)

### 2. Custom Domain (Optional)
- Triggers ‚Üí Custom Domains
- Add: `api.audiocity-ug.com`

### 3. Update Frontend
Change API URL in HTML files:
```javascript
const API_BASE_URL = 'https://audio-city-api-proxy.YOUR_SUB.workers.dev';
// Or with custom domain:
const API_BASE_URL = 'https://api.audiocity-ug.com';
```
