## Method 1: Using API Endpoint (Recommended)

curl -X POST https://api.audiocity-ug.com/api/admin/update-password \
  -H "Content-Type: application/json" \
  -d '{"email": "chilafrican@gmail.com", "password": "Semakulanico1"}'

Or use browser console:

fetch('https://api.audiocity-ug.com/api/admin/update-password', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    email: 'chilafrican@gmail.com',
    password: 'Semakulanico1'
  })
}).then(r => r.json()).then(console.log);
