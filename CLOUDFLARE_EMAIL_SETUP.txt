  CLOUDFLARE EMAIL ROUTING SETUP GUIDE
  Set up custom email addresses for audiocity-ug.com
═══════════════════════════════════════════════════════════════

✅ FREE - Cloudflare Email Routing is completely free
✅ Easy setup - Takes about 5-10 minutes
✅ Custom emails - Use @audiocity-ug.com addresses

═══════════════════════════════════════════════════════════════
  STEP 1: ACCESS EMAIL ROUTING
═══════════════════════════════════════════════════════════════

1. Go to Cloudflare Dashboard
   → https://dash.cloudflare.com

2. Select your domain: audiocity-ug.com

3. In the left sidebar, click "Email" → "Email Routing"

4. Click "Get started" button

═══════════════════════════════════════════════════════════════
  STEP 2: ENABLE EMAIL ROUTING
═══════════════════════════════════════════════════════════════

1. Cloudflare will show you DNS records to add
   - These are MX records for email routing
   - Cloudflare can add them automatically

2. Click "Add records" or "Enable Email Routing"
   - Cloudflare will add the required MX records

3. Wait 1-2 minutes for DNS propagation

═══════════════════════════════════════════════════════════════
  STEP 3: ADD DESTINATION ADDRESS
═══════════════════════════════════════════════════════════════

1. Go to "Email Routing" → "Destination addresses"

2. Click "Create address"

3. Enter your existing email:
   → chilafrican@gmail.com

4. Click "Save"
   - Cloudflare will send a verification email

5. Check your Gmail inbox
   - Open the verification email from Cloudflare
   - Click the verification link
   - Email routing is now active!

═══════════════════════════════════════════════════════════════
  STEP 4: CREATE CUSTOM EMAIL ADDRESSES
═══════════════════════════════════════════════════════════════

1. Go to "Email Routing" → "Routing rules"

2. Click "Create address"

3. Create custom email addresses:

   Example addresses you can create:
   ─────────────────────────────────
   • contact@audiocity-ug.com
   • support@audiocity-ug.com
   • info@audiocity-ug.com
   • admin@audiocity-ug.com
   • hello@audiocity-ug.com
   • nico@audiocity-ug.com

4. For each address:
   - Enter the custom address (e.g., "contact")
   - Select destination: chilafrican@gmail.com
   - Click "Save"

5. Done! Emails sent to these addresses will forward to:
   → chilafrican@gmail.com

═══════════════════════════════════════════════════════════════

1. Someone sends email to: contact@audiocity-ug.com
2. Cloudflare receives it

═══════════════════════════════════════════════════════════════
  SENDING EMAILS (OPTIONAL)
═══════════════════════════════════════════════════════════════

To send emails FROM your custom address:

1. Go to "Email Routing" → "Send from"

2. Click "Create address"

3. Enter: contact@audiocity-ug.com (or your preferred address)

4. Cloudflare will provide SMTP settings:
   - SMTP Server: smtp.cloudflare.com
   - Port: 587 (TLS) or 465 (SSL)
   - Username: (your Cloudflare email)
   - Password: (Cloudflare will generate)

5. Configure in your email client (Gmail, Outlook, etc.)

OR use Cloudflare's web interface to send emails directly.

═══════════════════════════════════════════════════════════════
  TROUBLESHOOTING
═══════════════════════════════════════════════════════════════

❌ Email not forwarding?
   → Check DNS records are correct
   → Wait 5-10 minutes for DNS propagation
   → Verify destination email is confirmed

❌ Can't verify destination email?
   → Check spam folder
   → Request new verification email
   → Make sure email address is correct

❌ Emails going to spam?
   → Set up SPF/DKIM records (Cloudflare does this automatically)
   → Wait 24-48 hours for email reputation to build
